## Key Terms

:::info Glossary Terms
- **Nav2**: ROS 2 navigation framework
- **Costmap**: 2D grid representing traversability
- **cuMotion**: GPU-accelerated motion planning library
- **Visual SLAM**: Simultaneous localization and mapping using cameras
