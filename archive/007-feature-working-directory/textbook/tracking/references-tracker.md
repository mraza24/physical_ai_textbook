## APA 7th Edition Format Examples

### Journal Article (with DOI)
```
Author, A. A., Author, B. B., & Author, C. C. (Year). Title of article. Title of Periodical, volume(issue), pages. https://doi.org/xxxxx
```

**Example**:
```
Mur-Artal, R., & Tardós, J. D. (2017). ORB-SLAM2: An open-source SLAM system for monocular, stereo, and RGB-D cameras. IEEE Transactions on Robotics, 33(5), 1255-1262. https://doi.org/10.1109/TRO.2017.2705103
```

### Conference Paper
```
Author, A. A., & Author, B. B. (Year). Title of paper. In Title of Conference Proceedings (pp. pages). Publisher.
```

**Example**:
```
Brohan, A., Brown, N., Carbajal, J., Chebotar, Y., Chen, X., Choromanski, K., ... & Zitkovich, B. (2023). RT-2: Vision-language-action models transfer web knowledge to robotic control. In Conference on Robot Learning (pp. 2165-2183). PMLR.
```

### Book
```
Author, A. A. (Year). Title of work (Edition). Publisher.
```

**Example**:
```
Craig, J. J. (2017). Introduction to robotics: Mechanics and control (4th ed.). Pearson.
```

### Website/Documentation
```
Organization. (Year). Title of page. URL
```

**Example**:
```
Open Robotics. (2023). ROS 2 documentation: Humble Hawksbill. https://docs.ros.org/en/humble/
```

### GitHub Repository
```
Organization or Author. (Year). Title of repository. GitHub. URL
```

**Example**:
```
Unity Technologies. (2023). Unity Robotics Hub. GitHub. https://github.com/Unity-Technologies/Unity-Robotics-Hub
```

### Preprint (arXiv, bioRxiv, etc.)
```
Author, A. A., & Author, B. B. (Year). Title of preprint. Name of preprint repository. URL or https://doi.org/xxxxx
```

**Example**:
```
Brohan, A., Brown, N., Carbajal, J., Chebotar, Y., Dabis, J., Finn, C., ... & Zitkovich, B. (2022). RT-1: Robotics transformer for real-world control at scale. arXiv preprint arXiv:2212.06817. https://arxiv.org/abs/2212.06817
```

## Reference Collection Guidelines

### When to Add a Reference

Add a source to this tracker when:
- You cite a technical claim or fact from an external source
- You reference official documentation (APIs, libraries, tools)
- You mention a research paper or algorithm
- You direct readers to external resources for further reading

### What NOT to Include

Do not add:
- General knowledge (e.g., "Python is a programming language")
- Internal cross-references (other chapters in this textbook)
- Uncited sources (only include what you actually reference)

### Multiple Authors

- **1-20 authors**: List all authors (use comma + `&` before last author)
- **21+ authors**: List first 19, insert ellipsis (...), then final author

**Example (21+ authors)**:
```
Smith, J., Jones, M., Brown, A., White, K., ... & Lee, S. (2023). Title. Journal, 10(2), 100-120.
```

### No Date

If publication date is unavailable, use `(n.d.)`:
```
Organization. (n.d.). Title of page. URL
```

### Retrieved Date

Only include retrieval date if content may change over time (e.g., wikis, preprints):
```
Wikipedia. (2023, November 15). Robotics. Retrieved December 11, 2024, from https://en.wikipedia.org/wiki/Robotics
```
